# Drawing Module - Quick Start Guide

## 5-Minute Tutorial

### Step 1: Start the Application
```bash
npm start
```

### Step 2: Wait for Chart to Load
- Ensure your data service is running
- Wait for the Gantt chart to appear

### Step 3: Enable Drawing Mode
1. Look for the button in the **top-right corner** of the chart
2. Click **"🖊️ Draw"**
3. Button changes to **"✏️ Drawing"** (now active)
4. Cursor changes to crosshair

### Step 4: Draw Your First Annotation
1. **Click and hold** on the chart
2. **Drag** to draw a line or shape
3. **Release** to complete the stroke

### Step 5: Change Colors (Optional)
1. Click the **🎨 color button**
2. Select a color from the palette
3. Draw again with the new color

### Step 6: Adjust Brush Size (Optional)
1. Use the **size slider** (shows "Size: 3" by default)
2. Move slider left (thinner) or right (thicker)
3. Try values from 1 to 20

### Step 7: Export Your Annotated Chart
1. Click **"📥 Export"** button
2. PNG file downloads automatically
3. Confirmation appears in chat panel
4. File saved as: `annotated-gantt-chart-[timestamp].png`

### Step 8: Clear Drawings (Optional)
- Click **"🗑️ Clear"** to remove all annotations
- This does NOT affect the chart data

### Step 9: Exit Drawing Mode
- Click **"✏️ Drawing"** button again
- Returns to normal chart interaction
- Drawings remain visible but can't be edited

## Tips & Tricks

### Smooth Drawing
- **Move slowly** for smooth lines
- **Move quickly** for straight lines
- **Small strokes** for detailed work
- **Large strokes** for emphasis

### Color Strategy
- 🔴 **Red**: Highlight problems or important areas
- 🟢 **Green**: Mark completed or good areas
- 🔵 **Blue**: General notes and annotations
- 🟡 **Yellow**: Areas needing attention
- ⚫ **Black**: General purpose drawing
- ⚪ **White**: Erase (draw over mistakes)

### Brush Size Guidelines
- **1-3 pixels**: Fine details, precise annotations
- **4-7 pixels**: Normal annotations, readable lines
- **8-12 pixels**: Bold emphasis, headers
- **13-20 pixels**: Very bold, backgrounds, covering

### Export Best Practices
1. **Wait** for chart to fully render
2. **Review** all annotations before exporting
3. **Test** export early to ensure quality
4. **Use** descriptive filenames (rename after download)
5. **Keep** original (export multiple versions)

### Workflow Recommendations

**Quick Annotations:**
1. Enter drawing mode
2. Draw quickly
3. Export immediately
4. Exit drawing mode

**Detailed Annotations:**
1. Study chart first
2. Plan what to highlight
3. Enter drawing mode
4. Draw methodically
5. Change colors as needed
6. Review before export
7. Export with confirmation

**Collaborative Sessions:**
1. Screen share the application
2. Draw while discussing
3. Get feedback on annotations
4. Export and send to participants
5. Clear and start fresh for next topic

## Keyboard & Mouse Guide

### Mouse Actions
- **Click + Drag**: Draw
- **Release**: Complete stroke
- **Click button**: Activate controls
- **Click outside palette**: Close color picker

### Visual Feedback
- **Crosshair cursor**: Drawing mode active
- **Normal cursor**: Drawing mode inactive
- **Button highlight**: Active/inactive state
- **Real-time drawing**: Paths appear as you draw
